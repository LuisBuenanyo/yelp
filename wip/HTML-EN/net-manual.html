Manually set network settings

If your network doesn't automatically assign network settings to your computer, you may have to manually enter the settings yourself. This topic assumes that you already know the correct settings to use. If not, you may need to ask your network administrator or look at the settings of your router or network switch.

To manually set your network settings:

  1. Open the Activities overview and start typing Settings.

  2. Click on Settings to open the control center.

  3. Click on Network to open the panel.

  4. In the left pane, select the network connection that you want to set up manually and click the button. For example, if you plug in to the network with a cable, click Wired. For a Wi-Fi connection, the button will be located next to the active network.

    Make sure that your wireless card is turned on or a network cable is plugged in.

  5. Click IPv4 and change Addresses to Manual.

  6. Type in the Address and Gateway, as well as the appropriate Netmask.

    These must be IP addresses; that is, they must be four numbers separated by periods (e.g. 123.45.6.78).

  7. In the DNS section, switch Automatic to OFF. Enter the IP address of a DNS server you want to use. Enter additional DNS server addresses using the + button.

  8. In the Routes section, switch Automatic to OFF. Enter the Address, Netmask, Gateway and Metric for a route you want to use. Enter additional routes using the + button.

  9. Click Apply. If you are not connected to the network, click the system status area on the top bar and connect. Test the network settings by trying to visit a website or look at shared files on the network, for example.
