Browse the web

  1. Move your mouse pointer to the Activities corner at the top left of the screen to show the Activities Overview.

  2. Select the Web browser icon from the bar on the left-hand side of the screen.

Alternatively, you can launch the browser by just typing web in the Activities Overview.

  1. Click the address bar at the top of the browser window and start typing in the website you want to visit.

  2. Typing in a website starts searching for it in the browser history and bookmarks, so you do not need to remember the exact address.

    If the website is found in the history or bookmarks, a drop-down list is shown below the address bar.

  3. From the drop-down list, you can quickly select a website using the arrow keys.

  4. After you have selected a website, press Enter to visit it.
