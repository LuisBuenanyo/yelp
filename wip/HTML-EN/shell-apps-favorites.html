Pin your favorite apps to the dash

To add an application to the dash for easy access:

  1. Open the Activities overview by clicking Activities at the top left of the screen

  2. Click the grid button in the dash and find the application you want to add.

  3. Right-click the application icon and select Add to Favorites.

    Alternatively, you can click-and-drag the icon into the dash.

To remove an application icon from the dash, right-click the application icon and select Remove from Favorites.
