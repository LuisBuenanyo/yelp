Make the keyboard cursor blink

If you find it difficult to see the keyboard cursor in a text field, you can make it blink to make it easier to locate.

  1. Open the Activities overview and start typing Keyboard.

  2. Click on Keyboard to open the panel.

  3. Select Cursor blinks in text fields.

  4. Use the Speed slider to adjust how quickly the cursor blinks.
